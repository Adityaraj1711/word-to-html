Question 1:

In the figure shown, if AD=BD and AE=EC, what is y+z in terms of x?

Option A:

90+x

Option B:

90-x

Option C:

90-x/2

Option D:

90+x/2

Correct Option:

C

Solution

As AD=BD => ∠DBA = ∠DAB = y

and AE = EC => ∠EAC = ∠ECA = z

thus in ΔABC ∠A + ∠B +∠C = 180

(y+ x + z) + y + z = 180

2(y +z) = 180 – x

y + z = 90 – x/2

Question 3:

In the figure shown, O is the centre of the circle. If ∠OBC=37° and ∠BAC=x°, what is the value of x?

Option A:

53°

Option B:

74°

Option C:

111°

Option D:

148°

Correct Option:

A

Solution

∠OBC = ∠OCB = 37° (∵ OB=OC)

∠BOC = 180° - (37°+37°) = 106°

∠BAC = x = ½(∠BOC) = ½(106°) = 53°
